HTML

HyperText Markup Language (Linguagem de Marcação de HiperTexto), é o bloco de construção básico da Web. É o que define a estrutura e o significado do conteúdo inserido na web.

"HiperTexto" são os links que conectam páginas da web entre si, seja dentro do próprio site ou outros sites, que são chamados de links externos. Isto é um link, normalmente eles tem a cor azul e são sublinhados, porém conseguimos alterar o estilo dos links através de outra tecnologia, chamada CSS.

HTML Semântico

Hoje em dia, a tecnologia HTML não serve apenas para estruturar o conteúdo na web, mas também informa o significado do conteúdo que está presente nos documentos através de tags semânticas. Dessa forma, fica mais claro tanto para desenvolvedores quanto para os browsers e outras engines processarem as informações.

Logo abaixo apresentamos as principais tags semânticas:

Tags

  • <header>
  • <section>
  • <article>
  • <nav>
  • <aside>
  • <main>
  • <figure>
  • <footer>
  • <a>
  • <strong>

header

Header é usado para representar o cabeçalho da página ou de uma seção. Dentro dessa tag podemos inserir vários elementos, para representar títulos, imagens, parágrafos ou até mesmo listas.
