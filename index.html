Complete Front-end Roadmap

Objetivo

O objetivo principal deste projeto, é estudar desde o básico até o avançado. A ideia surgiu após ver uma publicação no Instagram de um Roadmap completo para ser Front-end (Link Post). Logo, pensei em desenvolver um site onde terá todos os tópicos abordados na postagem e contendo também uma breve explicação sobre o assunto abordado, exemplos de como colocar em prática e quem sabe, gerando até alguns desafios. Dessa forma, poderei revisar conteúdo, praticar e disponibilizar para outras pessoas que estão começando.

O que é Front-end

Front-end é o desenvolvimento da interface gráfica do usuário de um site, utilizando HTML, CSS e Javascript. Com essas tecnologias os usuários podem visualizar e interagir com o site.

O que é HTML, CSS e Javascript?

Fazendo um breve resumo...

  • HTML é a parte de conteúdo, textos, imagens e etc.
  • CSS é a parte da beleza, são as cores dos textos, cores de fundo, bordas, sombras e entre outras características.
  • Javascript é a parte de interação com o usuário, o click de um botão, um evento, ao apertar uma tecla.
